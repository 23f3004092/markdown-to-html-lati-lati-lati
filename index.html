# Welcome to the Markdown Editor! This is a simple Markdown editor and viewer. You can type your Markdown content here and see it rendered below. * **Live preview:** See your changes instantly. * **Syntax Highlighting:** Code blocks are highlighted. ## Features - Tabbed interface for switching between source and rendered view. - `marked.js` for Markdown parsing. - `highlight.js` for code block syntax highlighting. ## Example Markdown ```javascript function greet(name) { console.log(`Hello, ${name}!`); } greet("World"); ``` This is a [link to GitHub](https://github.com). > Blockquote example.
